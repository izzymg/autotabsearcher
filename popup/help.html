Auto Tab Searcher
The selected tab will be reloaded at the set interval. When the term is found, it will sound an alarm. The alarm sound licensed under Creative Commons Attribution 4.0, and was obtained from: https://notificationsounds.com This add-on is open source and licensed under the GPLv3 license. You can obtain a copy of the source code by downloading the add-on from The Mozilla add-on site or the GitHub repository.


How to use this add-on:
  • "Tab to watch" lists all open tabs in your browser. Select the one you wish to monitor from this list.
  • "Search term" is the text (case-sensitive) you wish to search the page for each reload.
  • Regex means "regular expressions", an advanced format to search for text. Learn more here. By default, regex support is turned off, because some characters will fail to search unless properly escaped, for example the period '.'
  • "Reload interval" is the time in seconds waited after each reload of the page and search. It is recommended you use a value greater than 20 seconds. If the page loads slower than the reload interval, it may fail to search.
  • Finally, click "start" to begin monitoring the tab selected. You can monitor as many tabs as you like at once.
